# MMLリファレンス

TL;DR: OnGen の MML は PPMCK/MCK 流の音長指定（`C4` = 四分音符の C）と相対オクターブ（`>` / `<`）を正本とします。

## 音長とオクターブ

音符直後の数字は**音長**を表します。オクターブは `O` コマンド、または `>`（1 つ上げる）/ `<`（1 つ下げる）で指定します。

```text
PPMCK / MCK: c4 = 四分音符のC
OnGen:       C4 = 四分音符のC（オクターブは O / > / < で指定）
```

旧バージョンでは音符直後の数字をオクターブとして解釈していましたが、現在は廃止されています。

## コマンド一覧

| コマンド | 意味 |
|---------|------|
| `C D E` | ノート（音長省略時は L コマンドの値） |
| `C4 D8 E16.` | ノート + 音長指定（付点も併用可） |
| `O4` | オクターブ指定 |
| `>` / `<` | オクターブを 1 つ上げる / 下げる |
| `L8` | デフォルト音長 |
| `T120` | テンポ (BPM) |
| `V15` | 音量 (0〜15) |
| `R` / `R8` | 休符 |
| `C&C` | スラー（同じ音程の音符を連結） |
| `[CDE]3` | ループ（任意回数の反復。数字省略時は 2 回） |
| `@1` | 音色プリセット（1〜7） |
| `%12` | FM 変調指数 (0〜15) |
| `*4` | FM モジュレータ倍率（値÷2） |
| `~8` | LFO 深度 (0〜15) |
| `W(kick.wav)` | WAV サンプル再生 |
| `Q4` / `Q4,6` | ゲートタイム（音長の 4/8 を発音、第2引数は加算フレーム数） |
| `K+12` | トランスポーズ（半音） |
| `N32` / `N32,8` | 直接ノート番号（`n0` = C2、`n16` = C3、`n32` = C4）と音長 |
| `A`〜`E` | トラック宣言（1ファイル内の複数パート。行頭または `AB` 等の複数文字） |

独自拡張の `%`、`*`、`~`、`W(...)` は PPMCK/MCK 記法ではありません。

一致しないトークンは `ValueError` としてパーサーエラーになります（サイレントスキップしません）。

## PPMCK / MCK 互換性

| PPMCK / MCK記法 | 現状 | 備考 |
|-----------------|------|------|
| `c d e f g a b` | 部分対応 | 大文字・小文字を同一視 |
| `+`, `-` | 対応 | シャープ・フラット |
| `r4`, `r8` | 対応 | 休符の音長指定 |
| `t120` | 対応 | テンポ |
| `l4`, `l8` | 対応 | デフォルト音長 |
| `o4` | 対応 | オクターブ指定 |
| `v0`〜`v15` | 対応 | 固定音量 |
| `c4`, `c8` | 対応 | 音符ごとの音長指定 |
| `>`, `<` | 対応 | 相対オクターブ変更 |
| `c.`, `c4.` | 対応 | 音長指定と付点の組み合わせ |
| `&` | 対応 | 同じ音程の音符を連結 |
| `^` | 未対応 | タイの別表記（`&` を使用） |
| `q`, `Q` | 対応 | `q<num0>,<num1>`。音長 × `num0` / 分母 + `num1` フレーム。既定値は `q8,0`、分母は `#GATE-DENOM` で変更可 |
| `@q`, `@Q` | 対応 | フレーム単位の短縮（60fps 換算） |
| `[ ... ]4` | 対応 | `]` 直後の数字で反復回数 |
| `|: ... :|` | 未対応 | 多重リピート |
| `K<num>` | 対応 | 半音単位のトランスポーズ（-127〜126） |
| `n<num>,<len>` | 対応 | PPMCK 流の直接ノート番号。1オクターブを16値で表現し、12は未定義、13〜15は1オクターブ下のA〜B |
| トラック文字 `A`〜`E` | 対応 | 1ファイル内の複数トラック。曖昧な裸の音符列は単一トラックを優先。A/B=square、C=triangle、D/E=noise。`@0`〜`@3` は A/B のデューティ比 |
| `#TITLE` 等のヘッダ | 対応 | 行全体をスキップ（`;` 行コメントも同様） |
| `@v`, `@EN`, `@EP` 等のマクロ | 未対応 | エンベロープ・アルペジオマクロ |
| `@DPCM` | 未対応 | 代替として `W(...)` を使用 |

PPMCK/MCK 互換の対応状況は上表のとおりです（XinOne では Pyxel 方言 `--mml-dialect pyxel` は使用しません）。

## 実行例

```bash
uv run python tools/sound/sfx_generator.py --input "O4 L4 T120 C D E G" --style square -o output/mml/melody
```

ABC 記法は [BGMと楽譜](bgm-and-scores.md) を参照してください。
